Introducción a temas en Drupal 7

Cómo crear un tema

  1. Crear carpeta

    se recomienda situar el tema en 'sites/all/themes/custom/mitema'

    Idealmente el nombre del tema es el mismo que el de la carpeta.

  2. Crear esqueleto

    Crear un tema desde cero, clonar uno de fabrica por ejemplo Bartik o crear un subtema de de Omega por ejemplo

  3. Habilitar

    Realizar la habilitación en admin/appearance donde aparece un listado con todos los temas disponibles

Información del tema

Cada tema debe tener un archivo .info, en este caso mitema.info.


		name = Mi Tema Drupal 7
		description = Mi tema Drupal 7
		package = Core
		version = VERSION
		core = 7.x

Imagen del tema

Un tema puede tener una imagen visible en el administrador

path/to/theme/screenshot.png

Configurar tema

Dependiendo del tema, hay opciones de configuración disponibles como colores, funcionalidades, etc.

admin/appearance/settings/mitema

Estructura de archivos recomendada

Embed videos from your favorite online video service or with an HTML5 video element.


		mitema
		mitema/css // Estilos
		mitema/images // Imágenes usadas para el tema
		mitema/js // Archivos Javascript usados
		mitema/templates // Plantillas de diseño
